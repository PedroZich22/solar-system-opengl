## Getting Started

### Prerequisites

- CMake
- OpenGL
- A C++ compiler (e.g., GCC, Clang)

### Build and Run

To build and run the simulation, follow these steps:

```bash
cd build
cmake ..
cmake --build . && ../bin/main
```
